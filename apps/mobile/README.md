# apps/mobile

Expo / React Native app for **counter staff and managers**: live device grid, start/close sessions, take orders, walk-in sales, open/close shifts with cash reconciliation. Offline-first.

**Scaffolded in Phase 3 (walking skeleton).** This placeholder reserves the workspace slot. The `mobile-engineer` agent owns this app; it consumes `@ps/core` for all pricing/money/time logic.

Conventions: Arabic-first RTL, integer piastres, timers derived from timestamps (never `setInterval`). See root `CLAUDE.md`.
