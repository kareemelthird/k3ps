# apps/web

Next.js web app with two surfaces:

- **Owner dashboard** — pricing rules, products, devices, staff, reports & analytics, CSV export.
- **Super-admin portal** — tenant lifecycle (provision/suspend), support tools, (guarded) impersonation, billing.

**Scaffolded in Phase 3+ (owner dashboard in Phase 6, super-admin in Phase 7).** This placeholder reserves the workspace slot. The `web-engineer` agent owns this app; it consumes `@ps/core` and the same Supabase backend.

Conventions: Arabic-first RTL, integer piastres. See root `CLAUDE.md`.
